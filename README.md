# Expense Tracker (Трекер расходов)

**Автор:** [софия] [бородина]  
**Описание:** GUI-приложение на Python (Tkinter) для удобного учета и анализа личных финансовых расходов с возможностью фильтрации и локального хранения данных.

## Основные возможности
*   Добавление записей с автоматической валидацией числовых данных и формата даты.
*   Фильтрация сохраненных расходов по категориям и временным периодам.
*   Автоматический пересчет итоговой суммы расходов на основе активных фильтров.
*   Автономное сохранение и загрузка базы данных через JSON-файл (`expenses.json`).

## Как запустить приложение
1. Установите Python версии 3.x с официального сайта.
2. Скачайте файлы проекта или клонируйте репозиторий.
3. Откройте терминал в папке проекта и выполните команду:
   ```bash
   python app.py
   ```

## Инструкция по использованию
1.  **Добавление:** Введите число в поле "Сумма", выберите категорию, укажите дату в формате `ГГГГ-ММ-ДД` (по умолчанию стоит текущий день) и нажмите "Добавить расход".
2.  **Фильтрация:** В блоке фильтров выберите нужные критерии (категорию или диапазон дат) и нажмите "Применить".
3.  **Сброс:** Для отображения всех данных нажмите "Сбросить".

## Примеры тестовых сценариев
*   **Тест валидации суммы:** Попытка ввести в поле суммы текст "сто" или отрицательное число "-50" вызовет предупреждение.
*   **Тест валидации даты:** Ввод даты в формате "12.05.2026" вызовет ошибку. Правильный формат: "2026-05-12".
*   **Тест фильтра:** При указании диапазона с `2026-01-01` по `2026-01-31` в таблице останутся только январские траты, а счетчик "Итого" покажет сумму за этот месяц.
